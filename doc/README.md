## 폴더 구조

```
doc/
├─ 01-ERP-도메인-개론/      ← ERP가 무엇인지부터 시작하는 도메인 입문 자료
└─ 02-Phase-0-환경-구축/    ← Phase 0 학습 사이클의 산출물 (브리핑/설계/워크스루/시연)
```

## 읽는 순서 가이드

- **ERP가 처음이라면** → `01-ERP-도메인-개론/`을 1 → 2 → 3 순으로.
- **환경 구축이 끝나서 검증하고 싶다면** → `02-Phase-0-환경-구축/04-시연-가이드.md`만 따라 해도 충분.
- **코드를 자기 것으로 만들고 싶다면** → 코드 워크스루 1 → 9를 차례로 읽으며 실제 파일과 함께 비교.

다음 Phase 산출물(Phase 1 — 마스터 데이터)은 `03-Phase-1-마스터-데이터/`로 추가될 예정.
