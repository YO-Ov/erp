# ERP 학습 문서

ERP(전사적 자원 관리) 학습을 위한 도메인 정리 + Phase별 학습 산출물 모음.

## 폴더 구조

```
doc/
├─ 00-도메인-개론/             ← Phase 시작 전 ERP 도메인 워밍업
└─ 01-phase-0-환경-구축/       ← Phase 0 학습 사이클의 산출물 (브리핑/설계/워크스루/시연)
```

명명 규칙:
- **부모 폴더**: `00-`, `01-`, `02-` ... 커리큘럼 순서. 도메인 개론은 `00`, Phase 0은 `01`, Phase 1은 `02`, ... 식.
- **자식 파일/폴더**: `0-`, `1-`, `2-` ... 한 Phase 안에서의 7단계 사이클 순서 (한 자리 숫자).
- **손주 파일** (코드 워크스루 등): `01-`, `02-` ... 묶음 안에서의 일련번호 (두 자리).

부모/자식의 숫자 형식을 일부러 다르게 해서 어느 층의 번호인지 한눈에 구분되도록 함.

## 읽는 순서 가이드

- **ERP가 처음이라면** → `00-도메인-개론/`을 1 → 2 → 3 순으로.
- **환경 구축이 끝나서 검증하고 싶다면** → `01-phase-0-환경-구축/4-시연-가이드.md`만 따라 해도 충분.
- **코드를 자기 것으로 만들고 싶다면** → 코드 워크스루 1 → 9를 차례로 읽으며 실제 파일과 함께 비교.
