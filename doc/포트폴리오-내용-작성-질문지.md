# 포트폴리오 내용 작성 질문지

> `doc/포트폴리오-사이트-전략-및-구현계획.md` 의 §6(확인 필요 사항)을 답변 가능한 형태로 푼 문서.
> **작성법**: 각 질문의 `답변:` 뒤에 그대로 적으면 된다. **문장으로 다듬을 필요 없다** — 키워드·메모 수준이면 충분하고, 문장화는 내가 한다.
> **모르거나 기억 안 나면 `모름` 이라고 적고 넘어갈 것.** 빈칸으로 두면 "아직 안 봤다"인지 "없다"인지 구분이 안 된다.
>
> **우선순위**: 🔴 = 없으면 작업 못 함 / 🟡 = 있으면 훨씬 좋아짐 / ⚪ = 여유될 때
