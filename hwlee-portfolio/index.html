이런 걸 합니다

// domain

업무 도메인을 코드로

공공기관·물류·항공 ERP 현업에서 다져온 도메인 감각. 차변과 대변이 1원도 안 틀려야 하는 회계, 순서를 건너뛸 수 없는 생산 공정, 정해진 결재선 — 업무 규칙을 코드가 강제하도록 설계합니다.

// full-stack

백엔드부터 프론트까지

Java·Spring으로 API를 만들고, React·Vue와 TypeScript로 화면까지 직접 만듭니다. 서버 DTO와 프론트 타입을 1:1로 맞춰, 응답 구조가 어긋나면 컴파일 단계에서 걸리게 했습니다.

// ops

설계부터 배포까지

Docker·Caddy·Cloudflare로 실제 도메인에 올리고, 푸시하면 자동 배포되는 CI까지. 만드는 데서 끝내지 않고 무중단 배포·비용 최적화까지 직접 운영합니다.
